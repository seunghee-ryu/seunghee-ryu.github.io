# 테스트 자동화와 Mocha



## 테스트는 왜 해야 하는가?

- 코드를 수동으로 재실행 하면서 테스트를 하면 무언가를 놓치기 쉽다.
- 테스팅 자동화는 테스트 코드가 실제 동작에 관여하는 코드와 별개로 작성되었을 때 가능하다. 테스트 코드를 이용하면 함수를 다양한 조건에서 실행해 볼 수 있는데, 이때 실행 결과와 기대 결과를 비교할 수 있다.
