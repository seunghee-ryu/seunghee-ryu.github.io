# JDBC

## Statement vs PreparedStatement

### Statememt
- SQL 문법 + 값 -> SQL 문장 -> DBMS 프로토콜에 맞춰 가공 -> 가공된 SQL 문장 -> 전송 -> DBMS와 통신

### PreparedStatement
- SQL 문법 -> SQL 문장 -> DBMS 프로토콜에 맞춰 가공 -> 가공된 SQL 문장 + 값 -> 전송 -> DBMS와 통신
