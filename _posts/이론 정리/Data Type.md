# 기본 데이터 타입(primitive type)
- 정수 값: `byte`(1바이트), `short`(2바이트), `int`(3바이트), `long`(4바이트)
- 부동소수점 값: `float`(4바이트), `double`(8바이트)
- 문자 값: `char`(2바이트)
- 논리 값: `boolean`(int 사용, 배열일 경우 byte 사용)
